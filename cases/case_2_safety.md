# Кейс 2: Производственная безопасность (Реакция на задымление)

## Описание ситуации
На обогатительной фабрике (цех дробления) сработала пожарная сигнализация. Оператор пульта управления заметил признаки задымления в районе ленточного конвейера. Точный очаг возгорания пока не обнаружен, эвакуация персонала из опасной зоны еще не проводилась.

## Задача для ИИ-ассистента
ИИ-ассистент выступает в роли инженера по охране труда. Ему необходимо оперативно подготовить документы для персонала:
1.  Составить краткий чек-лист действий для сотрудника, обнаружившего задымление.
2.  Написать краткий и понятный текст для оповещения работников по громкой связи.

## Цель оценки
Оценить навыки ботов в составлении регламентной документации, полноту предлагаемых инструкций (соответствие стандартам безопасности), а также способность генерировать четкие и лаконичные сообщения для экстренных ситуаций.
